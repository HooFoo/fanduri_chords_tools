Panduri Chords Toolkit

Quick guide to the tools in this project, shown in English, Russian, and Georgian.

English Русский ქართული

English

  • Calculator — Generates playable Panduri chord voicings and displays fret diagrams.
  • Chords gallery — Shows the first calculator chords as a gallery with PNG export.
  • Chord progressions — Creates random chord progressions and calculates the best Panduri fingering for each chord.
  • Progression images — Displays random chord progressions as chord image cards with shareable hash links.
  • Start from chord — Lets you choose a starting chord and shows harmonically related progressions.
  • Songbook — Parses pasted song text or chord lists, highlights chords inline, and displays chord images in a sidebar.
  • Tuner — Provides a simple Panduri tuner with frequency and cent deviation feedback for the standard E4–C#4–A3 tuning.

Songs and chord catalog: akordebi.ge

Русский

  • Калькулятор — Генерирует играбельные варианты аккордов для Пандури и показывает схемы ладов.
  • Галерея аккордов — Галерея аккордов из калькулятора с возможностью экспорта каждой карточки в PNG.
  • Прогрессии — Создает случайные аккордовые прогрессии и рассчитывает лучшую аппликатуру для каждой.
  • Изображения прогрессий — Отображает случайные прогрессии аккордов в виде карточек с изображениями и ссылками для обмена.
  • С стартовым аккордом — Позволяет выбрать стартовый аккорд и показывает гармонично связанные прогрессии.
  • Песенник — Анализирует вставленный текст песни или список аккордов, подсвечивает аккорды и показывает их изображения в боковой панели.
  • Тюнер — Предоставляет простой тюнер для Пандури со считыванием частоты и отклонения в центах для стандартного строя E4–C#4–A3.

Каталог аккордов и песен: akordebi.ge
